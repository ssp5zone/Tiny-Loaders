AlmondJS - Manual Bundle

AlmondJS is a small part of RequireJS, which has no provision to download dependencies on demand. Hence it only supports bundled modules.

Also no configuration required.

As of now, most amd-module bundlers (Rollup, js-bundler, amd-bundler, Parcel.js) suck. (No no. I am not wrong. Hence, there must be some issue with plugin, Node.js, Browser, Computer, Electricity, Gravity, Space or Time)

Hence manually bundled them in one file.

You can write your own bundler as well. All you have to do for bundling is,

  1. Go to said file.
  2. Check if it has a define(). If not shim it (wrap it).
  3. If the define() does not have an id, provide one based on the path. Otherwise there would be no way to identify a block.
  4. Copy the contents in the final build file. (order in not important)
  5. Update all reference to this file to the newly given id.
  6. Loop through its dependencies.forEach(Goto 1)

There is an added advantage of bundling. Double-click and run the file. No need of server as no lazy loading.
